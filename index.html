Welcome to my home page


My name is Arindam Mazumdar , and I am post doctoral fellow at IIT Kharagpur, India


My research interest is mainly focused on cosmology. In the sector of late time cosmology I have worked on Neutrino Cosmology, Structure formation, Spherical Collapse, Constraining theoretical models form cosmological observations. I have also worked on some topics of Early Universe Cosmology like Inflationary Cosmology, Reheating, Preheating, Non-gaussianity in Cosmic Microwave Background.

The following codes are written by me and my collaborators:


2loop

(https://github.com/arindam-mazumdar/2loop)


This code calculates the 3rd order (two loop) matter power spectrum given a linear matter power spectrum. It is a combination of C-code and Mathematica files



RSD-BiSpec

(https://github.com/arindam-mazumdar/rsd-bispec)


This code calculates the multipoles of large scale structure bispectrum in redshift space. This code can calculate the multioples for bispectrum generated by primordial nongaussianity as well as gravitationally induced non-gaussianity up to second order in perturbation theory. This is written in Python.
